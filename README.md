# Blog App

Blog app is a web application for writing and reading bloggs.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
// Initialize npm
npm init

// Download dependencies: express, pug
npm install express
npm install pug
```

## Usage

```javascript
node app.js
```
